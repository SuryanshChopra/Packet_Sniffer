# Packet_Sniffer
A python program to sniff out the data flowing through the computer
### Requirements:
-> scapy_http 
```
pip install scapy_http
```
### Note: 
* Before Initiating the code make sure to run the `arp_spoof.py` before `packet_sniffer.py`
### Initiate:
```
python packet_sniffer.py 
```
